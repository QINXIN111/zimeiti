### 🔧 技术栈

- **后端**: Python 3.8+ + FastAPI + uvicorn
- **前端**: 原生 HTML/JS/CSS
- **自动化**: Playwright (Chromium)
- **内容生成**: OpenAI 兼容 API (DeepSeek/Qwen/GPT)
- **数据存储**: YAML 配置文件 + 内存任务队列

### 🚀 快速启动

```bash
# 安装依赖
pip install -r requirements.txt

# 安装 Playwright 浏览器
playwright install chromium

# 启动 Web 服务
uvicorn web.app:app --host 0.0.0.0 --port 8888
```

访问 http://localhost:8888
